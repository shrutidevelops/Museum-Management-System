# Museum Management System

## 📖 Description
A Python-based application designed to manage museum artifacts, visitor records, and ticket bookings efficiently. The system helps in organizing different categories like stone, sculpture, textiles, and pottery while also handling memberships and ticket searches.

## 🚀 Features
- Artifact management (stone, sculpture, textiles, pottery)
- Museum membership system
- Ticket booking system (basic, combo, drama, jewellery)
- Search functionality for tickets
- Structured modular code

## 🛠 Technologies Used
- Python
- HTML (for templates)
